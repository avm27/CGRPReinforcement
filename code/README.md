# RNA-Seq Pre-Processing Pipeline

**Project:** CGRPReinforcement
**PI / Lab:** Luis Tuesta Lab
**Genome:** mm10 (NCBI RefSeq)

---

## Overview

This directory contains all bash scripts needed to reproduce the RNA-seq pre-processing pipeline from raw FASTQ files to a single merged gene count matrix. Three independent datasets are processed and then combined into one final output used for all downstream analyses.

The pipeline runs in four sequential stages:

```
Step 0  →  Download raw FASTQ files (SRA / GEO)
Step 1  →  Pre-process LT2015 + LT2019 single-end data   (01_preprocessing_LT2019_LT2015_merged.sh)
Step 2  →  Pre-process PBN2 paired-end data              (02_preprocessing_PBN2.sh)
Step 3  →  Generate merged gene count matrix              (03_generate_gene_count_matrix.sh)
```

> Steps 1 and 2 are independent of each other and can be run in any order or in parallel.  
> Step 3 must only be run after **both** preprocessing steps have completed.

## Key Parameter Differences Between Datasets

> **Important:** LT2015 and LT2019 are single-end libraries. PBN2 is paired-end.
> Do not use the wrong preprocessing script for the wrong dataset.

| Parameter | LT2015 / LT2019 (Script 02) | PBN2 (Script 03) |
|-----------|-----------------------------------|-------------------|
| Read layout | Single-end | **Paired-end** |
| trim_galore mode | `--dont_gzip` | `--gzip --paired` |
| Trimmed output files | `*_trimmed.fq` | `*_val_1.fq.gz` + `*_val_2.fq.gz` |
| STAR `--readFilesCommand` | *(not set — uncompressed input)* | `zcat` |
| STAR `--readFilesIn` | Single file | R1 file + R2 file |

## Requirements

### Software Dependencies


| Tool | Purpose | Versions |
|------|---------|---------|
| `trim_galore` + `cutadapt` | Adapter trimming and quality filtering | (v.0.6.7) (v.1.8) |
| `STAR` | Splice-aware alignment to mm10 | (v.2.7.10a) |
| `samtools` | BAM manipulation, filtering, indexing | (v.1.9) |
| `stringtie` + `prepDE.py` | Transcript assembly and gene count matrix generation | (v.2.1.7) |
| `pigz` | Parallel gzip compression of trimmed FASTQs | (v.2.6) |
| `python`| Necesary for StringTie | (v.3.7.8) |

### Reference Genome Files

Scripts look for mm10 reference files in `~/genome/mm10/` by default. Override any path
by setting environment variables before calling a script:

```bash
mm10genomedir=/your/path/STAR_index \
mm10gtf=/your/path/mm10.ncbiRefSeq.gtf \
mm10fai=/your/path/mm10.fa.fai \
bash 01_preprocessing_LT2019_LT2015_merged.sh
```

Required files:
- **STAR genome index** — built from mm10 NCBI RefSeq FASTA using your installed version of STAR
- **mm10.ncbiRefSeq.gtf** — NCBI RefSeq annotation GTF for mm10
- **mm10.fa.fai** — samtools FASTA index for mm10

## Step-by-Step Instructions

### Step 0 — Download raw FASTQ files

**LT2015 + LT2019** (for script 01):
```bash
bash 00_download_LT2015_LT2019.sh
```

**PBN2** (for script 02):  
Download paired-end fastq files from GEO accession **GSE324173** and place them in
`data/raw/PBN2/fastq/`. Expected naming convention:
```
<sampleName>_R1_001.fastq.gz
<sampleName>_R2_001.fastq.gz
```

---

### Steps 1–2 — Pre-process each dataset

These two scripts are independent and can be run in any order or simultaneously.

```bash
bash 01_preprocessing_LT2019_LT2015_merged.sh
bash 02_preprocessing_PBN2.sh
```

Each script will remind you when it finishes:
```
Next step: once ALL datasets are processed, run:
  bash 03_generate_gene_count_matrix.sh
```

---

### Step 3 — Generate the merged gene count matrix

Run only after **both** preprocessing scripts have completed successfully.

```bash
bash 03_generate_gene_count_matrix.sh
```

This script:
1. Collects StringTie GTF files from all three dataset directories
2. Handles both SRA (`<acc>.fastq.gz`) and PBN2 (`<sample>_R1_001.fastq.gz`) naming conventions automatically
3. Builds a temporary sample list, runs `prepDE.py`, then deletes the sample list
4. Writes the final output to `data/gene_count_matrix_LT19_PBN2_LT15_Merged.csv`

## Outputs

### Per-dataset (scripts 01–03)

Each preprocessing script produces the following under `data/raw/<experimentname>/`:

| Directory | Contents |
|-----------|----------|
| `trimmed_fastqs/` | Compressed trimmed FASTQ files |
| `StringTie/` | Per-sample GTF files — input to script 04 |
| `FPKM/` | Per-sample FPKM tables |
| `Logs/` | STAR alignment logs and trim_galore reports |

Intermediate BAM files are removed after StringTie quantification to conserve disk space.

### Final merged output (script 03)

| File | Location | Description |
|------|----------|-------------|
| `gene_count_matrix_LT19_PBN2_LT15_Merged.csv` | `data/` | Merged raw gene counts across all samples — input to DESeq2 - analysis/index.Rmd |
